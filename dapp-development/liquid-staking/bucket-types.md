# Bucket Types

Three bucket types are supported in the current implementation:

| Amount    | Duration (days) | Duration (blocks) |
| --------- | --------------- | ----------------- |
| 10,000    | 91              | 1,572,480         |
| 100,000   | 91              | 1,572,480         |
| 1,000,000 | 91              | 1,572,480         |
